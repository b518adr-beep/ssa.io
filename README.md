# التقنية والأناقة — بدر الشافعي

الموقع الشخصي لبدر الشافعي: مطور ويب ومصمم واجهات.

## هيكل الملفات
- `index.html` — صفحة الموقع
- `style.css` — التنسيقات
- `script.js` — قسم الأعمال ولوحة الإدارة والتأثيرات
- `images/` — صور الأعمال

## إضافة عمل جديد بشكل دائم
افتح `script.js` وعدّل قائمة `myWorks` في أعلى الملف (موضّحة بتعليقات عربية)،
وضع صورة الغلاف داخل مجلد `images/` ثم اكتب مسارها مثل: `images/work-4.jpg`

## النشر على GitHub Pages
ارفع الملفات إلى المستودع، ثم من Settings → Pages فعّل النشر من فرع main.
